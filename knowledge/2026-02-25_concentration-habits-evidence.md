## 要点

集中力・持続的注意力を高める習慣を「エビデンス信頼度(40%)×効果量(30%)×取り入れやすさ(30%)」の3軸でスコアリング。最高スコアは戦略的カフェイン摂取（4.7）と十分な睡眠（4.4）。睡眠は「基礎インフラ」であり、これなしには他の介入効果が大幅に減少する。

## 詳細

### 上位スコアの習慣

| 習慣 | スコア | 核心 |
|---|:---:|---|
| **戦略的カフェイン摂取（100-300mg）** | 4.7 | EFSA承認の健康強調表示。75mgで注意力向上。即効性・低コスト |
| **7-9時間の睡眠確保** | 4.4 | 睡眠不足でd=0.87の注意力低下。6時間×2週間=2日徹夜相当 |
| **定期的な有酸素運動** | 3.8 | 実行機能にd=0.68。BDNFを増加、海馬体積2%増加 |
| **パワーナップ（10-30分）** | 3.7 | NASA研究: 26分昼寝で警戒度54%向上。10分が最も即効性あり |
| **概日リズムの安定化** | 3.4 | 不規則な睡眠は学業成績低下と関連。±30分以内が理想 |
| **認知作業前の急性運動** | 3.4 | 実行機能にd=0.20-0.35。運動後15-60分が最も効果的 |
| **ウォーキング休憩（10分）** | 3.3 | 60-90分ごとの中断。取り入れやすさが最高（5点） |
| **筋力トレーニング** | 3.1 | SMD 0.40-0.53（全体認知機能）。有酸素運動との併用推奨 |

### カテゴリ別の特徴

- **睡眠**: 集中力の「基礎インフラ」。カフェインは就寝8時間前までに停止
- **運動**: 有酸素運動が最もエビデンス豊富。「朝の運動→集中作業」ルーティンが効果的
- **栄養**: カフェインが最も確実。地中海式食事は長期的な認知保護
